# r-anime-page-bg

A userstyle that sets a maximum width for content on /r/anime and displays a background of your choice underneath.

## Installation

### With [Stylus](https://add0n.com/stylus.html)

Stylus is a browser extension for Chrome and Firefox that lets you load custom website styles easily. To install with this extension, all you need to do is click below:

**[:package: Install the style](https://raw.githubusercontent.com/Geo1088/r-anime-page-bg/master/main.user.css)**
